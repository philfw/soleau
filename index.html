Poor Man's Copyright

The Soleau envelope is named for the French inventor Eugene Soleau. It is a way of proving when you created an original work. Simply seal your original plan or work into a letter, then send the letter to yourself, using the postal service's postmark as proof of the date.

Soleau is a modern analogue to the Soleau envelope. Copy and paste an original work into the box below, and that file will be emailed directly to you with a unique ID number. Simultanesouly, that ID will be tweeted out with the hashtag #Soleau. ⬇
